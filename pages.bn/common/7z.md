# 7z

> উচ্চ স্পীড অনুপাতে ফাইল সংরক্ষণকর্তা।
> আরও তথ্য পাবেন: <https://manned.org/7z>।

- নতুন বা বিদ্যমান আর্কাইভে একটি ফাইল বা ডিরেক্টরি যোগ করুন:

`7z a {{আর্কাইভ.7z/এর/পাথ}} {{ফাইল_অথবা_ডিরেক্টরি/এর/পাথ}}`

- বিদ্যমান আর্কাইভকে এনক্রিপ্ট করুন (ফাইলের নাম সহ):

`7z a {{এনক্রিপ্টেড.7z/এর/পাথ}} -p{{পাসওয়ার্ড}} -mhe=on {{আর্কাইভ.7z/এর/পাথ}}`

- মূল ডিরেক্টরি স্ট্রাকচার বজায় রেখে আর্কাইভটি আলাদা করুন:

`7z x {{আর্কাইভ.7z/এর/পাথ}}`

- বিশেষ ডিরেক্টরি এইচআউটপুটের সাথে সংরক্ষণ থেকে নেওয়া:

`7z x {{আর্কাইভ.7z/এর/পাথ}} -o{{আউটপুট/এর/পাথ}}`

- `stdout` এর জন্য একটি সংরক্ষণ থেকে বাহাতে:

`7z x {{আর্কাইভ.7z/এর/পাথ}} -so`

- একটি নির্দিষ্ট সংরক্ষণ প্রকার ব্যবহার করে সংরক্ষণ করুন:

`7z a -t{{7z|bzip2|gzip|lzip|tar|zip}} {{আর্কাইভ/এর/পাথ}} {{ফাইল_অথবা_ডিরেক্টরি/এর/পাথ}}`

- একটি সংরক্ষণের সামগ্রী তালিকা বদ্ধ করুন:

`7z l {{আর্কাইভ.7z/এর/পাথ}}`
